  • Command Pattern in Game Development
    • This application is built in Unity using a Command Pattern to seperate the logical and visual states of the game.

    • Link to final result: conraddudziak.github.io/TurnBasedCommandPattern
  • Image Generation with Genetic Algorithms
    • A genetic algorithm written in JavaScript that receives a target image and converges a population of shapes to appear similar to the image.
    • Link to final result:conraddudziak.github.io/ImageGeneration
    • Link to final report:conraddudziak.github.io/pdf/CapstoneFinalReportDudziak
  • Cellular (Unity Game)
    • Cellular is a turn-based strategy game created by a team of four software engineers including myself.
    • The objective in cellular is to eliminate your opponent by equipping your cells with weapons.
      1. Movement Prototype
        • Features the creation of a hexagonal map and the pathfinding of units.
        • Link to final result:conraddudziak.github.io/CellularMapMovement
      2. Equipment Prototype
        • Features the ability of cells to equip items.
        • Link to final result:conraddudziak.github.io/CellularEquippableItemsPrototype
      3. Visualizing Movement Prototype
        • Features the seperation of visual and logical concerns by implementing visual movement on top of previously existing logical movement.
        • Link to final result:conraddudziak.github.io/CellularVisualMovementPrototype
      4. Action Points Prototype
        • Features the implementation of action points which limit the amount of actions a cell can perform each turn.
        • Link to final result:conraddudziak.github.io/CellularActionPointsPrototype
      5. Item Spawning Prototype
        • Features items spawning on the map and the ability for cells to equip the items that they navigate to.
        • Link to final result:conraddudziak.github.io/CellularItemSpawningPrototype
      6. Player Vs AI Prototype
        • Features the ability to have a human player play against a basic AI. (Turn can be passed with spacebar)
        • Link to final result:conraddudziak.github.io/CellularPlayerVsAIPrototype
      7. AI Vs AI Prototype
        • Demonstrates the Command Pattern application as two AI's logically complete the game within seconds, but all visual events are enqeued to occur with time.
        • Link to final result:conraddudziak.github.io/CellularAIVsAIPrototype
      8. Early Game Demo
        • An early demo of the game with an early attempt at a UI and an eased user experience.
        • Link to final result:conraddudziak.github.io/CellularDemo
      9. Pre-alpha
        • The version of the game play tested by individuals other than the developers.
        • Link to final result:conraddudziak.github.io/CellularPre-alpha
      10. Alpha
        • The first iteration of implemented user feedback.
        • Link to final result:conraddudziak.github.io/CellularAlphaBuild
      11. Beta V1
        • The second iteration of implemented user feedback with an overhaul of the games user interface.
        • Link to final result:conraddudziak.github.io/CellularBetaV1
      12. Final
        • The latest iteration of the game with changes made to balancing and match length.
        • Link to final result:conraddudziak.github.io/CellularBetaV2

Code Repositories of Distributed Systems and Cloud Integration

  • NASA API Web App
    • A web application that uses Microsoft Azure to fetch, store, and display data from the NASA API. Services used include App Services, Logic Apps, Blob Storage, Azure Tables, and Telemetry.
    • Link to source code repository: github.com/ConradDudziak/NASA-API-Web-App
  • HTTP Web Crawler
    • A web crawler created as a class assignment which finds the first http link in a web page, and continues to crawl until the number of hops is achieved.
    • Link to source code repository: github.com/ConradDudziak/HTTP-Web-Crawler
  • UDP Framework in .Net and Unity
    • This application uses UDP to pass messages between clients and an authoritative server. Based off work done by Sloan Kelly.
    • Link to source code repository: github.com/ConradDudziak/UDPFramework
  • Continuous Integration with Azure Pipelines
    • An example repository of an Azure Web Service that is tested and deployed automatically when code is pushed.
    • Link to source code repository: github.com/ConradDudziak/ContinuousIntegration
